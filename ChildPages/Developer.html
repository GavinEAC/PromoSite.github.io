Why Did I Make This?

The Console Arcade is my personal starting point in C++ game development. I figured it would be best to learn the fundamentals of the language through console applications, instead of learning computer graphics at the same time.

Originally, I was learning to make games with python, using the pygame library, but Python is a interpreted language and makes running bigger programs hard. So I switched C++ for it's faster processing, as well as its closer control over hardware. The reason I'm sharing these games on a website, is that I need to make a Promotional website for my computer science class.
